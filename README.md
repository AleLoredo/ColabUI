# ColabUI

Una extensión de Chrome diseñada específicamente para **optimizar la interfaz de Google Colab**, maximizando el espacio vertical y mejorando la productividad.

## Características

*   **Optimización de UI**: Minimiza barras de herramientas, reduce márgenes y ajusta el tamaño de fuente para ver más código.
*   **Persistencia**: Guarda tu preferencia (Activado/Desactivado) para recordar tu configuración en futuras visitas.
*   **Barra de Estado Plegable**: Incluye un botón tipo "marcador" para expandir o contraer la barra de estado a voluntad.
*   **Botones Laterales**: Mueve los indicadores de "Variables" y "Terminal" a la barra lateral para un acceso rápido.

## Instalación

1.  Clona o descarga este repositorio.
2.  Abre Google Chrome y ve a `chrome://extensions`.
3.  Activa el **Modo de desarrollador** (esquina superior derecha).
4.  Haz clic en **Cargar descomprimida**.
5.  Selecciona la carpeta de este proyecto (`ColabUI`).

## Uso

1.  Abre cualquier notebook de Google Colab.
2.  La extensión aplicará tu última configuración guardada automáticamente.
3.  Haz clic en el icono de la extensión para abrir el popup:
    *   Usa el botón **Toggle On / Toggle Off** para activar o desactivar las optimizaciones instantáneamente.
4.  Usa el botón en la esquina inferior derecha para ver la barra de estado completa (minimizar/maximizar).
